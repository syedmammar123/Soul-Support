Soul Support is a web application aimed at providing mental health support and assistance to those in need. 


Tech Stack:
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Firebase
- Payment Gateway: Stripe

Installation Steps:

1. Clone the repository:
    git clone https://github.com/syedmammar123/Soul-Support.git

2. Frontend Setup:
    a. Navigate to frontend folder: cd soul-support/frontend
    b. Create .env file using .env.example as a template.
    c. Run npm install to install dependencies.
    d. Start the frontend server: npm run dev

3. Backend Setup:
    a. Open a new terminal and navigate to backend folder: cd soul-support/backend
    b. Create .env file using .env.example as a template.
    c. Run npm install to install dependencies.
    d. Start the backend server: npm start

Ensure MongoDB is connected locally or configure a remote MongoDB connection in the .env file.
